# Bentukin - Website VRChat Model Store

## Cara Edit & Upload

### 1. Edit data model
Buka `models/data.json` — ganti:
- name, price, description
- preview3d: path ke file .vrm
- images: path ke foto
- gumroad/booth: link checkout kamu

### 2. Upload file 3D (VRM/GLB)
Taruh file `.vrm` atau `.glb` di folder `models/`
Contoh: `models/model-1.vrm`
Lalu update path di `models/data.json`

### 3. Upload foto
Taruh foto `.jpg` atau `.png` di folder `images/`
Contoh: `images/model-1-1.jpg`
Lalu update path di `models/data.json`

### 4. Upload ke GitHub
Setelah edit, push semua file ke repo:
https://github.com/mumininmuh-alt/mumininmuh-alt.github.io
